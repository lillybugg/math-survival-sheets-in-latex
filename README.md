Welcome! This repo is your one-stop stash for **Math Survival Sheets**—all the essential formulas, tips, and cheats for surviving math courses. Perfect for Algebra, Trig, Calc and more!

## What's Inside
- **LaTeX source files**: Edit, remix, or learn from the code.  
- **PDF versions**: Ready-to-use sheets for quick reference.

- Subjects: Foundations of Algebra, Trigonometry, and Calculus 1. More on the way!

## How to Use
1. Browse the PDFs in the `/pdfs` folder for quick studying.  
2. Want to tweak or build your own? Check the `.tex` files in `/latex`.  
3. Compile with LaTeX (`pdflatex yourfile.tex`) if you want a fresh PDF.
